# Gravity Runner

Gravity Runner is a fast-paced Unity platformer featuring **gravity inversion mechanics**, momentum-driven gameplay, and polished UI/UX design. Built with modular architecture and scalable scoring systems, it blends dynamic mechanics with professional documentation for both learning and play.

---

## 🎮 Features
- **Gravity Inversion**: Flip gravity to overcome obstacles and master momentum.
- **Responsive Controls**: Smooth player movement and collision handling.
- **Scalable Systems**: Inspector-based tuning, persistent managers, and multi-level progression.
- **Polished UI/UX**: Custom scoreboards, fonts, and dynamic interface elements.
- **Documentation**: Academic-style project report with integrated diagrams.
